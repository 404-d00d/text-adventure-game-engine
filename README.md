# Text Adventure Game Engine

Text adventure game, except with 3D-style game controls.

# How To Use

Download the repository, make sure all Python files stay in the same folder, then run:

python DemoGame.py

# Controls

- `w` Move Forward
- `a` Move Side Left
- `s` Move Backward
- `d` Move Side Right
- `q` Turn Left
- `e` Turn Right
- `f` Interact With The Tile In Front Of You
- `F` Interact With The Tile You Are Standing On
- `i` Open Inventory

You can also enter several commands in one line, such as:

wweff
